# Mycelium Maze
## Описание
Прототип игры в жанре roguelike/soulslike.
## Управление
- W/A/S/D - ходьба

- E - взаимодействие с костром

- R - использование лечебной фляги

- ЛКМ - атака

- ПКМ - блок
